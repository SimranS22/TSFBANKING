# TSFBANKING - A BASIC BANKING SYSTEM

The 'Basic Banking System' is a Web-Development Internship Project (Task #1) as part of GRIP(Graduate Rotational Internship Program)@The Sparks Foundation

## Check out the website here 
<https://tsf-website-grip.000webhostapp.com/index.php>

## Description
1. Project is built using :
  * Front-end technologies : HTML, CSS, Javascript, Bootstrap
  * Back-end technologies : PHP
  * Databases : MySQL
  * IDE : VSCode
  
2. Databases contains 2 tables : 
  * `Customer`: Maintains All Customer details
  * `Transact`: Mantains All Transaction details
  
## Task 
To create a simple dynamic website which has the following:
1. Dummy data in database for upto 10 customer.
2. Customer table will have basic fields such as name, email, current balance, etc. Transact table will record all transfers that occurred, along with their date and time.
3. Main flow of the Website: Home Page > View all Customers > Select and View one Customer > Send Money > Select Customer to transfer to > View all Customers > View Transaction History.
